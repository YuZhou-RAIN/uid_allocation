# `獨一符識` 模塊  
具普適之性，可納諸存檔而共用。

## 功用
- 授玩家以寰宇獨一之識數

## 用術
### 初行
施此令一度：
```mcfunction
function 5c0ef3bb-e60e-450e-ae8d-1aae812416d5/init
```

### 布設
置循環令塊，延遲無刻，使恒執此令：
```mcfunction
function 5c0ef3bb-e60e-450e-ae8d-1aae812416d5/main
```

### 卸除
施此令一度：
```mcfunction
function 5c0ef3bb-e60e-450e-ae8d-1aae812416d5/deinit
```
既施，則削所用計分板之目\
而後移此行為包即可。
